#### 💫 About Me:
Saya  seorang Web Developer.<br>Saya mengembangkan CarbonStock, sebuah aplikasi berbasis website aplikasi ini digunakan  untuk menghitung dan monitoring cadangan karbon sesuai standar SNI. Aplikasi ini telah digunakan di 24 dari 25 lokasi survei di Indonesia pada tahun 2025 dan mendukung penelitian SDGs Center Telkom University dalam memantau keberlanjutan lingkungan.<br>Saya memiliki pengalaman  magang di Direktorat PuTI Telkom University, mengembangkan produk TI Non Akademik sebagai Back end Developer.<br>Meskipun memiliki kondisi cerebral palsy (gagap dan organ kanan kaku), saya tetap mampu bekerja secara profesional, beradaptasi dengan tim, dan memberikan kontribusi optimal.Saya percaya bahwa tantangan adalah kesempatan untuk tumbuh, dan saya berkomitmen untuk terus belajar serta mengembangkan diri di bidang teknologi.<br>Saat ini, saya terbuka untuk kesempatan kerja full-time maupun freelance di bidang Back End Development, dan siap berkontribusi dalam pengembangan sistem backend yang stabil, efisien, dan dapat digunakan secara nyata.<br>
